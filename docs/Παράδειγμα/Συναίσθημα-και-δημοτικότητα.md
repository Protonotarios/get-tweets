
# Συναίσθημα σε σχέση με τη δημοτικότητα


Η δημοτικότητα ενός tweet εκφράζεται με δύο τρόπους, 

1.  με τον αριθμό των **retweet** (αναδημοσιεύσεων), ή/και
2.  με τον αριθμό των **αγαπημένων** (like) που έχει λάβει.
